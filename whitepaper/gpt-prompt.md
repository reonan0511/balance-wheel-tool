# ChatGPT-5.5 Pro へのプロンプト

下記をコピペして ChatGPT-5.5 Pro に投げてください。
そのあと `whitepaper-source.md` の中身を続けて貼り付けます。

---

## プロンプト本体（コピペ用）

```
以下のMarkdown原稿から、コーチ向けの実務ホワイトペーパー（電子書籍/PDF配布物）を
PowerPointの.pptxファイルとして作成してください。

【ドキュメント仕様】
- 形式: .pptx（PowerPoint 16:9 ワイドスクリーン）
- ページ数: 表紙含めて約30ページ
- 言語: 日本語
- 用途: コーチが業務時間外に読む実務ガイド + クライアント向け配布ツールの使い方解説

【デザイン方針】
- 「イケてる資料感」を最優先
- ミニマル＆モダン、余白たっぷり、上質なビジネスドキュメント感
- 配色は GPT 推奨のクリーンなパレットで OK（特定ブランドカラー指定なし）
- フォント: 日本語は游ゴシック / Noto Sans JP 系、英数は Inter / Helvetica 系
- 各章扉ページにはセクション番号とタイトルを大きく配置
- 表紙は印象的なヒーロー画像 + タイトル + サブタイトル
- 各ページに余白とリズムを持たせる（情報詰め込みすぎNG）

【画像について】
- [IMAGE: 〜] と書かれた箇所には、その説明に沿った画像を生成してください
  （GPT-image-2 or DALL-E でその場で生成 → 配置してください）
- [SCREENSHOT: 〜] と書かれた箇所は、別途 PNG ファイルを後から差し込みます。
  その位置に「ここに [SCREENSHOT: 〜] を配置」というプレースホルダーを置いてください
- 表紙のヒーロー画像は特に印象的に。コーチング/対話/ウェルビーイングを連想させる
  抽象的かつ温かみのあるイラストレーション

【構造ルール】
- 「---」で区切られた各セクションを1ページ（または論理的に近い場合2ページ）に対応させる
- # = 表紙タイトル
- ## (PART X) = 各PARTの扉ページ
- ## (X章) = 各章のタイトルページ（必要なら2〜3ページに分割可）
- 表は表として、リストはリストとして、引用は引用として、適切な書式で表現
- コードブロック（```で囲まれた部分）はメッセージテンプレなので、見やすい枠で表示

【出力】
完成した .pptx ファイルを「balance-wheel-handbook.pptx」として書き出してください。

以下が原稿です:

[ここに whitepaper-source.md の中身を全部貼る]
```

---

## 使い方

1. 上記のプロンプト本体（` ``` ` で囲まれた部分）をコピー
2. ChatGPT-5.5 Pro の新しい会話で貼り付け
3. 続けて `whitepaper-source.md` の中身を全部貼り付け
4. 送信
5. GPT が .pptx を生成 → ダウンロード
6. PowerPoint / Keynote / Google Slides で開いて微調整

## 想定される追加プロンプト（必要に応じて）

**もし表紙が物足りないとき**:
```
表紙のヒーロー画像をもう一段上品に。
「コーチがクライアントに新しい視点を渡す瞬間」を象徴する
抽象的でフォトリアル寄りのイラストにしてください。
朝日が差し込む静謐な雰囲気で。
```

**もし全体的に文字が多すぎるとき**:
```
各ページの文字量を30%減らし、ビジュアルウェイトを増やしてください。
箇条書きは「3つまで」を原則に、それ以上は別ページに分割してください。
```

**もし配色が硬すぎるとき**:
```
配色を「ウェルネス・ライフスタイル誌」風にリファイン。
ベージュ・モカ・くすみブルー・ソフトグリーンなど、温かみのあるトーンへ。
```

**特定のページだけ作り直したいとき**:
```
○○ページ（タイトル: ○○）だけ作り直してください。
他は変更不要。今のスタイルに揃えてください。
```

## 後工程

GPT が出した .pptx に対して:

1. ツール画面のスクショを差し込む（[SCREENSHOT: 〜] のプレースホルダー位置）
2. URL リンクが正しく機能するか確認
3. 著者プロフィール・連絡先の情報を最新化
4. PDF として書き出し
5. WordPress の eBook ダウンロードフォームに登録
